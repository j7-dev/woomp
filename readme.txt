=== 好用版擴充 MorePower Addon for WooCommerce ===
Requires at least: 4.9.8
Tested up to: 5.7.2
Requires PHP: 7.0
License: GPLv3

Empty plugin to demonstrate the WP Plugin Updater.

== Description ==

WooCommerce 好用版擴充，改善結帳流程與可變商品等區塊，讓 WooCommerce 更符合亞洲人使用習慣。
